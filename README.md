# Array Analyzer

This is a simple JavaScript project that analyzes an array of numbers entered by the user.

## Features:
- Takes user input as comma-separated values (e.g., 1,2,3,4)
- Converts input string into an array
- Calculates:
  - Sum of all numbers
  - Maximum (largest) number
- Displays result instantly

## Concepts Used:
- Arrays
- Loops (for loop)
- String methods (split)
- Array methods (map)
- Number conversion
- DOM manipulation

## How it Works:
- User enters values like: 1,2,3,4
- Input is split into an array using `split(",")`
- Converted into numbers using `map(Number)`
- Loop is used to calculate sum and find max value
- Result is displayed on screen

## Example:
Input: 1,2,3,4  
Output: Sum: 10 | Max: 4
